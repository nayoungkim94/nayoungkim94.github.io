2024
------
**Towards Fair Language Modeling via Parameter-Efficient Methods by Machine Feedback**
* Mitigation of social biases in large language models (T5, BERT, LLaMA 2) for toxicity and hate-speech detection.
* Trained LLMs to learn fairness and reduce bias using reinforcement learning combined with parameter-efficient tuning (LoRA, P-tuning).

**MEGAWATT: MAST for Evaluating Generative AI in Worker–Automation Team Tasks**
* Applied the MAST trust-assessment framework to evaluate baseline performance and inform appropriate adoption of GPT-4 in intelligence and analysis (I&A) tasks.
* Conducted human-subject studies to measure whether off-the-shelf or improved outputs lead to appropriate use, including correct rejections.
* Improved GPT-4 response quality with prompt engineering and retrieval-augmented generation (RAG) for summarization, NER, and conversation tasks.

**Automated Evaluation of Machine-generated Summaries using RLHF**
* Trained an LLM-based classifier to evaluate document–summary pairs through multi-class classification and reinforcement learning with a handcrafted human-preferences dataset.
* Conducted expert evaluations on output scores to validate the proposed learning method.

2023
------
**PADTHAI-MM: Designing Trustworthy, Human-Centered AI Systems Using the MAST Methodology**
* Developed a novel AI design framework addressing the challenge of designing trustworthy AI systems.
* Demonstrated effectiveness through development of an AI-enabled decision support system that positively impacted user trust perceptions.
* Conducted association analysis between participant ratings and trust-impacting information, providing a theoretical basis for the framework.

2022
------
**READIT: Reporting Assistant for Defense and Intelligence Tasks**
* Trained a Transformer-based text-summarization system for intelligence analysis.
* Built a user-friendly web interface using Node.js and Google Cloud Platform, allowing analysts to access summarized reports.

**Facewise: AI-based Face ID Verification System**
* Engineered a face ID verification system for security screening, ensuring reliable and efficient identity authentication.
* Implemented face-matching with CNNs and ResNet, fine-tuning model parameters to optimize performance.

2021
------
**Interpreting Text Classifiers with Counterfactual Explanation**
* Final project for CSE 472 (Social Media Mining).
* Implemented counterfactual models for a multi-layer neural network used in text classification.

2017
------
**Biomedical Entity Relation Extraction**
* Extracted biomedical entities and identified relations using the Comparative Toxicogenomics Database (CTD) via distant supervision.
* Implemented and trained a tree-RNN model (SPINN) combined with a word–character embedding model.
